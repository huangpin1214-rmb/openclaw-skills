# BIS EAR 中国法规时间线（2018-2026）

## 2018年
- ECRA (Export Control Reform Act) 生效，确立BIS法律框架
- 对华管制相对宽松

---

## 2019年
| 日期 | 事件 |
|------|------|
| **2019.05** | 华为被列入实体清单（Entity List）<br>原因：国家安全/外交政策<br>影响：所有美国技术出口需许可证 |
| **2019.06** | 多家中国高校/科研机构被列入 |
| **2019.08** | 曙光/海光等超算企业被列入 |

---

## 2020年
| 日期 | 事件 |
|------|------|
| **2020.05** | 华为升级管制<br>使用美国技术需获得许可证 |
| **2020.08** | 进一步限制华为获取芯片 |
| **2020.12** | 中芯国际被列入实体清单<br>原因：军事最终用户风险 |
| **2020** | 军事最终用户(EMS)管制加强 |

---

## 2021年
| 日期 | 事件 |
|------|------|
| **2021.04** | 7家中国超级计算机企业被列入 |
| **2021.07** | 多个中国量子/航空航天企业被列入 |
| **2021.11** | 中国气球/监视技术相关企业被列入 |

---

## 2022年（关键年份）
| 日期 | 事件 |
|------|------|
| **2022.02** | 俄乌战争，BIS扩大对俄管制 |
| **2022.03** | 俄罗斯全面制裁 |
| **2022.10.07** | **历史上最严格管制**<br>1. 先进计算芯片管制 (A100, H100)<br>2. 半导体制造设备 (SME)<br>3. 超级计算机<br>4. **"推定拒绝"政策**<br>5. 外国直接产品规则(FDP)扩大适用范围 |
| **2022.10** | 长江存储被列入实体清单 |
| **2022.12** | 数十家中国企业被列入 |

---

## 2023年
| 日期 | 事件 |
|------|------|
| **2023.02** | 中国气球事件，相关企业被列入 |
| **2023.03** | 28家中国企业被列入（量子/航空航天） |
| **2023.05** | 22家中国量子企业被列入 |
| **2023.06** | 43个实体/50个条目被列入（31个中国）<br>原因：人权侵犯、弹道导弹、解放军飞行员培训 |
| **2023.10.17** | **两项重大临时规则**<br>1. SME IFR（半导体制造设备）<br>2. AC/S IFR（先进计算/超级计算机）<br>针对中国半导体产业全面限制 |
| **2023** | 累计355个中国实体被列入 |

---

## 2024年（最严格年份）
| 日期 | 事件 |
|------|------|
| **2024.01** | 2家中国激光雷达企业被列入 |
| **2024.04.04** | 澄清规则发布<br>回答行业问题，明确管制范围 |
| **2024.04.11** | 11个实体被列入（6个中国）<br>原因：支持解放军现代化、援俄 |
| **2024.05.09** | 37个中国实体被列入<br>22个量子相关、11个气球相关、4个军事现代化 |
| **2024.10** | 多轮实体清单更新 |
| **2024.12.02** | **历史上最大规模行动**<br>1. 140个实体清单新增<br>2. 14个实体清单修改<br>3. 针对中国半导体产业全面升级<br>4. ECAD/TCAD软件管制 |
| **2024** | 中国实体清单数量创历史新高 |

---

## 2025年
| 日期 | 事件 |
|------|------|
| **2025.01.13** | **AI扩散管制框架**<br>1. 控制先进计算芯片<br>2. 控制AI模型权重<br>3. License Exception AI Authorization (AIA)<br>4. Universal VEU (UVEU) |
| **2025** | 继续扩大管制，但重点转向AI/新兴技术 |
| **2025** | 中国实体累计超过400个 |

---

## 2026年（政策松动）
| 日期 | 事件 |
|------|------|
| **2026.01** | 无人机出口管制简化 |
| **2026.02** | **H200、MI325X等芯片对华出口**<br>审查政策从"推定拒绝"转为"逐案审查" |
| **2026.02** | 商务部将20家日本实体列入出口管制 |
| **2026.03** | BIS Federal Register 暂无重大更新 |

---

## 关键政策演变总结

```
2018-2020: 试探性管制
     ↓
2021: 逐步升级
     ↓
2022.10: 全面爆发（推定拒绝）
     ↓
2023-2024: 最严格时期
     ↓
2025-2026: 局部松动（逐案审查）
```
