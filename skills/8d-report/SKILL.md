# 8D Report Generator

8D 问题分析报告生成工具，专为芯片 Fabless 公司设计。

## 8D 步骤说明

| 步骤 | 名称 | 内容 |
|------|------|------|
| D0 | 紧急措施 | 临时围堵措施，防止问题扩散 |
| D1 | 团队组建 | 成立 8D 团队，明确职责 |
| D2 | 问题描述 | 详细描述问题现象、影响范围、发现时间 |
| D3 | 临时措施 | 采取的临时纠正措施 |
| D4 | 根本原因分析 | 5Why、鱼骨图、故障树分析等 |
| D5 | 长期措施 | 永久性纠正措施 |
| D6 | 实施与验证 | 措施实施计划及验证结果 |
| D7 | 预防措施 | 预防类似问题再次发生 |
| D8 | 团队认可 | 团队总结与关闭 |

## 使用方式

```bash
# 生成 8D 报告
python3 8d-report/scripts/generate_8d.py --issue "客户投诉：芯片上电后无输出"

# 或使用模板
python3 8d-report/scripts/generate_8d.py --template basic --output 8D_Report.docx
```

## 输入信息

生成报告前需要准备：
- 客户名称
- 产品型号/批次
- 问题描述
- 发现时间
- 影响范围（数量、金额）
- 初步分析

## 输出格式

- Word 文档 (.docx)
- 包含所有 8D 步骤的完整模板
- 符合行业标准的格式
