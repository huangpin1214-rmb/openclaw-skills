# BIS CCATS Expert

BIS EAR CCATS 申请专家 - 芯片出口管制分类分析 Skill

## 功能

- ECCN 分类分析 (3A001/3A090/3A991/5A002/5A992/EAR99)
- CCATS 申请可行性评估
- 一致性检查
- 英文申请书草案生成
- 中文管理摘要
- 风险评估

## 使用场景

用户需要分析芯片产品是否需要申请 CCATS 时使用此 Skill。

## 输入要求

请提供：
- 产品名称/型号
- 主要功能
- 接口/协议
- 是否包含计算单元 (CPU/GPU/NPU/DSP/FPGA)
- 制程节点
- 封装类型
- 是否支持 HBM/Chiplet
- 是否为数据中心/AI/HPC 设计
- 加密功能
- 当前怀疑的 ECCN
