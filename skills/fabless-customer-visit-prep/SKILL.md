# Fabless客户拜访准备

帮助芯片Fabless公司的销售/FAE/质量人员在拜访客户质量部门前做好充分准备。

> 💡 **快速使用：** 一旦说有客户拜访，直接说"准备XXX客户拜访"，我会给你完整的拜访方案 + Checklist 自检

## 📋 快速 Checklist

```
⏰ 拜访前1周
- [ ] 确认时间地点
- [ ] 确认参与人员
- [ ] 准备产品资料

⏰ 拜访前1-3天
- [ ] 了解客户背景
- [ ] 准备问答
- [ ] 确认产品导入情况（已导入/正在导入/发货量）

⏰ 拜访当天
- [ ] 带齐物品
- [ ] 提前到达

⏰ 拜访后
- [ ] 发送总结
- [ ] 跟进事项

⚠️ 必检问题
- [ ] 客户导入哪些产品？
- [ ] 已导入还是正在导入？
- [ ] 已导入产品是否发生过客诉？
- [ ] 主力产品发货量多少？

🔬 质量数据必检
- [ ] 产品认证（ AEC-Q100 / ISO）
- [ ] 可靠性数据（PPM、TTFF）
- [ ] 历史客诉情况
- [ ] 客诉响应时间要求
- [ ] PPM目标指标
```

> 📄 完整版 Checklist：`~/.openclaw/workspace/templates/customer-visit-checklist.md`

## 核心问题（必须询问）

每次拜访客户质量部门时，必须主动询问并记录以下信息：

### 1. 产品导入情况
> "请问目前我们哪些产品已经在贵司导入使用了？"

- 具体产品型号
- 应用场景
- 月度/年度用量预估
- 采购渠道（直采/代理）

### 2. 历史问题查询
> "请问前期我们产品在贵司使用过程中，有出现过什么问题吗？"

- 问题类型（功能不良、可靠性、供货等）
- 发生时间
- 处理状态（已解决/处理中/未解决）
- 客诉单号（如有）

---

## 补充问题（根据需要询问）

### 🔍 产品与项目
- 今年有哪些新项目需要选型？
- 预计月度/年度用量是多少？
- 项目启动/量产时间节点？

### ⚠️ 质量相关
- 贵司对我们产品有哪些特定的质量要求？（AQL、PPM等）
- 需要提供哪些可靠性测试报告？（如AEC-Q100）
- 对客诉响应和处理的时间要求？

### 📊 竞争与市场
- 目前有使用其他供应商的芯片吗？对我们有什么评价？
- 是否有替换现有供应商的计划？

### 🔧 技术支持
- 需要哪些技术支持？（FAE现场、远程、样品等）
- 需要哪些技术文档？（datasheet、可靠性报告、应用笔记等）

### 💼 商务与供应
- 价格有没有目标区间？
- 对交期、库存有什么要求？
- 目前通过哪家代理商采购？

### 💼 商务与供应
- 价格有没有目标区间？
- 对交期、库存有什么要求？
- 目前通过哪家代理商采购？

### 🔎 合规与环保
- 需要符合哪些环保标准？（RoHS、REACH等）
- 有没有原产地要求？

## 🎯 客户关心的交流内容

每次拜访前，必须了解客户当前最关心的话题：

### 常见客户关注点
| 类别 | 典型问题 |
|------|----------|
| 供货保障 | 产能是否充足？交期多久？ |
| 质量可靠性 | 失效率PPM？可靠性测试报告？ |
| 价格竞争力 | 能否降价？目标价多少？ |
| 技术支持 | 是否提供FAE支持？响应时间？ |
| 国产化需求 | 是否需要国产替代？ |
| 资质认证 | AEC-Q100/101/200？ISO认证？ |
| 配合度 | 样品支持？紧急订单响应？ |
