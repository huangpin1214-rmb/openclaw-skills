# 初中物理错题本

专业的初中物理错题管理与分析工具。

## 功能

1. **错题收集** - 录入错题题目、答案、解析
2. **错误分类** - 区分知识漏洞、粗心大意、思路错误
3. **原因分析** - 分析错误类型，给出针对性建议
4. **相似题推荐** - 根据错题知识点推荐类似题目
5. **复习提醒** - 根据记忆曲线提醒复习
6. **统计报告** - 统计各章节错误率，生成学习报告

## 知识点覆盖

### 八年级上册
- 机械运动（长度测量、速度计算）
- 声现象（声音的产生、传播、特性）
- 物态变化（温度、熔化、凝固、汽化、液化）
- 光现象（光的直线传播、反射、折射）
- 透镜及其应用

### 八年级下册
- 力（力、弹力、重力、摩擦力）
- 运动和力（牛顿第一定律、二力平衡）
- 压强（固体压强、液体压强、大气压强）
- 浮力（浮力、阿基米德原理）
- 功和机械能（功、功率、机械效率）

### 九年级
- 机械运动（电压、电流、电阻）
- 欧姆定律
- 电功率
- 焦耳定律
- 磁现象（磁场、电流的磁效应）
- 电磁感应
- 能源与可持续发展

## 错误类型分类

| 类型 | 描述 | 建议 |
|------|------|------|
| 知识漏洞 | 知识点未掌握 | 回归教材，夯实基础 |
| 思路错误 | 解题方法不对 | 归纳解题模型 |

## 输出示例

### 错题记录

```
📝 错题记录

题目：一辆汽车以 20m/s 的速度匀速行驶，当汽车刹车后以 5m/s² 的加速度减速，求刹车后 4s 内的位移。

❌ 错误答案：80m

✅ 正确答案：80m（正确）

分析：
- 错误类型：计算错误
- 刹车时间：v = at → t = 20/5 = 4s
- 4s 时刚好停下，位移 s = vt - ½at² = 20×4 - 0.5×5×16 = 80-40 = 40m
- 注意：汽车刹车问题要先判断停车时间
```

## 常用解题模型

### 力学
- 受力分析：先重力、后弹力、最后摩擦力
- 平衡问题：找准研究对象，正交分解
- 浮力计算：F浮 = ρ液gV排

### 电学
- 串并联识别：电流路径法
- 欧姆定律：I = U/R，串联分压，并联分流
- 电功率：P = UI = I²R = U²/R

### 光学
- 反射定律：反射角等于入射角
- 折射规律：空气中的角最大
- 凸透镜成像：物距像距关系
