## 一、核心概念速查

### 1. 实体清单 (Entity List)
- 定义：列入可能违反美国国家安全/外交政策的实体
- 管理：ERC委员会（商务部牵头，国防/国务院/能源/财政部参与）
- 决策：列入需多数票，移除需全票

### 2. 脚注含义
| 脚注 | 含义 |
|------|------|
| Footnote 1 | 军事最终用户 (744.22) |
| Footnote 3 | 与被列入实体关系导致推定拒绝 |
| Footnote 4 | 涉及国家安全，更严格审查 |

### 3. 许可证审查政策
| 政策 | 含义 |
|------|------|
| 推定拒绝 (Presumption of Denial) | 原则上拒绝，极少批准 |
| 逐案审查 (Case-by-Case) | 根据具体情况审批 |
| 批准 (Approval) | 原则上批准 |

### 4. De Minimis 规则
- 外国产品中美技术含量低于阈值则不受管制

### 5. FDP规则 (外国直接产品规则)
- 使用美国技术生产的外国产品受管制

## 三、CCL分类 (Export Control Classification Number)

### 0-9类
- 0: 核相关
- 1: 材料/化学
- 2: 材料加工
- 3: 电子
- 4: 计算机
- 5: 电信/信息安全
- 6: 传感器/激光
- 7: 导航/航空
- 8: 船舶
- 9: 航空航天/推进

### 600系列 (国防相关)
- 600系列：国安物品

### 重大案例

|------|------|----------|------|
